# Housing Regression (R)

Мини-проект на R для регрессии цены жилья на датасете `housing.csv`.

## Что делает
- Загружает датасет `DeepLearning/dataset/housing.csv`
- Делает сплит train/test (80/20)
- Обучает линейную регрессию `lm`
- Считает метрики RMSE, MAE, R²
- Сохраняет модель в `model.rds`

## Запуск
Из папки проекта:

```bash
Rscript run.R
```

## Примечания
- Используется только базовый R (без внешних пакетов).
- Если в CSV есть индексный столбец, он автоматически удаляется.
